# 🤖 AI-Generated Image Detection

A Deep Learning project to classify images as **Real** or **AI-Generated** using **MobileNetV2 Transfer Learning** with TensorFlow/Keras.

---

## 📌 Project Overview

This project aims to detect whether an image is real or generated by AI.

The model is built using MobileNetV2 with Transfer Learning and Fine-Tuning to achieve high classification performance.

---

## 🚀 Features

- Image Classification (Real vs AI-Generated)
- MobileNetV2 Transfer Learning
- Fine-Tuning
- Data Augmentation
- Early Stopping
- Model Checkpoint
- Reduce Learning Rate on Plateau
- Confusion Matrix
- Classification Report
- Gradio Web Interface

---

## 🛠️ Technologies Used

- Python
- TensorFlow
- Keras
- MobileNetV2
- NumPy
- Matplotlib
- Scikit-learn
- OpenCV
- Gradio
